# ai-based-fault-detection-in-power-electronic-converters
Developed an intelligent fault detection system for DC-DC power converters using machine learning.
Simulated buck converter under multiple fault conditions (open switch, short circuit, load variation) in MATLAB/Simulink
 Collected voltage & current waveform dataset for normal and faulty conditions
 Performed feature extraction and dataset preprocessing using Python (NumPy, Pandas)
Trained classification models to automatically identify converter faults
Achieved automated condition monitoring and predictive maintenance capability
Tech Stack: Python, Machine Learning, MATLAB/Simulink, Signal Processing
